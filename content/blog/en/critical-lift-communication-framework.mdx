## Command structure

The lift director shall verify a single command source prior to movement authorization. Critical lifts require strict readback and hold-point confirmations.

<TechnicalNote>
  <p>Communication reliability is an engineered risk control, not an administrative step.</p>
</TechnicalNote>

## Required protocol

<ProcedureBox>
  <ol>
    <li>Assign command authority and backup roles.</li>
    <li>Validate radio clarity and channel isolation.</li>
    <li>Use concise standardized command phrases.</li>
    <li>Apply stop/hold confirmation at each critical transition point.</li>
  </ol>
</ProcedureBox>

<InspectionPanel>
  <ul>
    <li>Radio battery status verified prior to lift sequence.</li>
    <li>Blind lift checkpoints mapped to physical reference points.</li>
    <li>Signal and rigger positions confirmed before load travel.</li>
  </ul>
</InspectionPanel>
